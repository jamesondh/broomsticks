Good question. I need to make a tutorial. For now, here are the basic steps:

1) Unzip a skin pack (using Winzip or pkunzip) 2) go into directories that were unzipped 3) edit images (textures) using your favorite paint program 4) run the game (it will pick up new skins from the directories) 5) when you are done, zip the directories back up into a zip file and send it to me.

  • Comments: "Paul, problem... again... but it's about skins. How do I change it? The skins11.zip can't change back into a "File"... If this makes no sense, just recommend anything..."

    To edit a skin you first have to unzip all the files in the .zip. Then use any photoediting software to make your changes to the skin. When your done, zip all those files back together and make sure that .zip file isn't named the same thing as any other Broomstick 3D file. To use this skin while playing the game you have to make sure that it has the ".zip" at the end of its file name and that all other skin files do not. To remove the ".zip" from a file name just rename the file. Then when you run the game your new skin should be applied.

    Hope that covers anything. I haven't made a skin in a long time so if I missed something sorry.
